The Grocery List App General Description and Design:

This phone app is designed to make grocery shopping easier by allowing users to collaboratively edit the list. You first have to enter some login information to a database. Then, you can use the app. The list of groceries can be made once logged in! This is done by entering text into a text area. The text is stored in a linked list. See pictures of the design below:
